Данный скрипт создан при помощи Isetup 5.5.0 для создания 
инсталяционного пакета Sigma.

В папке {app} - расположены исходные файлы програмы, чтобы
создать новую версию файлы необходимо распологать в данной папке

В папке embedded - файлы ресурсов для инсталяционного пакета

В папке Output - isetup размещает собранный пакет
